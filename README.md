# Jelly Jumper Sample

Physics settings inspired by Super Mario World 

![jelly jumper sample](./jelly-jumper.gif)

## Setup

```bash
npm install
npm start
```

## Controls

Move - Arrow keys

Run - S

Jump - A

Also works with gamepads.
